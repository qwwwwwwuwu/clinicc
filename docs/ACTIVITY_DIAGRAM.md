1 Пациент:
-  Отправляет POST /appointments/
- Заполняет:
     * ФИО
     * ID врача
     * Время приёма

2 Система:
   - Проверяет:
     * Занято ли время у врача (SQL-запрос)
     * Корректность временного интервала
   - Создаёт запись
   - Возвращает ID

3 Ошибки:
   - 400: Время занято
   - 422: Невалидные данные
